# E2E 验收清单

## 桌面端

1. 打开首页，确认顶部英雄区、市场区、求购快览、运营看板均可见。
2. 搜索任意关键词（如“雅思”），结果列表变化，搜索记录写入个人中心。
3. 点击书卡进入详情页，执行收藏与分享操作。
4. 在详情页打开智能聊天，输入“多少钱/在哪里面交/要了”，出现协议卡片。
5. 通过确认流程完成下单，个人中心“我购买的书”出现新增记录。

## 求购流程

1. 进入求购页，查看求购列表。
2. 发布一条新求购，返回列表可见新增项。
3. 打开某条求购详情，点击“响应并进入对话”。
4. 在对话中触发“要了”或“确认成交”，验证闭环提示。

## 个人中心

1. 收藏页能看到已收藏书籍。
2. 我求购的书页能看到自己发布的求购信息。
3. 我购买的书页能看到已完成的订单记录。
4. 建议与反馈可提交，提交后列表出现新记录。
5. 搜索记录可展示并清空。

## 移动端

1. 页面可滚动且布局不溢出。
2. 侧边导航折叠为顶部顺序布局。
3. 详情页、求购页、个人中心可正常交互。
